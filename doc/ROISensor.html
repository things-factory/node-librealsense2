Class: ROISensor

ROISensor()

Sensor for managing region of interest.

Constructor

new ROISensor()

Construct a ROISensor object, representing a RealSense camera subdevice The newly created ROISensor object shares native resources with the sensor argument. So the new object shouldn't be freed automatically to make sure resources released only once during cleanup.
